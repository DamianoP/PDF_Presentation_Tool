Personal Data Collection

This Web Application does not collect personal data from users.

All presentation files opened through the Web Application are processed locally in the user's browser and are not uploaded to any server.

Remote Control and Question and Answer Features (WebSocket Communication)

The Web Application includes optional features that allow a smartphone to control a presentation opened on another device (for example a desktop computer) or another feature that allows the audience to ask questions to the presenter.

When these features are used, the devices communicate through a temporary WebSocket connection established with the server.

This communication transmits minimal control messages (for example "next slide" or "previous slide") required for the remote control functionality or messages related to the questions and answers feature.

No personal data, files, or presentation content are transmitted or stored by the server during this process. The connection exists only for the duration of the active session. All the data related to the questions and answers feature are deleted after the end of the session and are not stored by the server.

Cookies

The Web Application may use a technical session cookie strictly necessary for the correct operation of the service.

This cookie does not track users, does not contain personal information, and is used only to maintain the current session.

Server Logs

Like most websites, the server hosting this website may automatically record basic technical information in server logs.

These logs may include information such as IP address, browser type, request time, and requested resources. This information is used exclusively for security, debugging, and operational purposes and is not used to identify users.

Third Parties

The Web Application does not share user data with third parties and does not use third-party tracking or analytics services.
